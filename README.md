# Python Datatypes and Operations Demo

## Description
This program is created to practice basic Python concepts. It shows how to:
- Use different data types
- Perform arithmetic operations
- Convert user input using type casting
- Handle invalid input using try-except
- Concatenate strings and numbers
- Understand dynamic typing in Python

## File Included
- `datatypes_demo.py` – Main Python program
- `task2output.png` – output screenshot

## How to Run

1. Open terminal in the project folder  
2. Run:
python datatypes_demo.py
